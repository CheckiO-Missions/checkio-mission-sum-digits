The maximum number of one digit is 9. So you need to repeat operations, until the num becomes less or equal 9. For such kind of task while loop is optimal. 

while num > 9:
    ...
Ok. Now help me to start converting the given number.

As I said before, you need to convert you current number into string format first. Do it with built-in str() function. 

str(num)

string is interpreted as a sequence, so now you may iterate over it and do operations with each character.

How to convert each character back into integer?

The easiest way is to use built-in map() function with arguments int and the string from previous hint. 

map(int, str(num))

This converts each character of string into int format and returns all as map object. Notice, that there are no parenthesis near int. Its because you refer to the function, not calling it.

I feel its almost done, but need a hint more.

Now you have a sequence of digits as map object. The cool thing is, that you may use this object as argument of built-in sum() function without any further conversion! 

sum(map(int, str(num)))

It's actually your new num so assign the result of calculations to the variable inside while loop.

So, how my final solution should look like?

You just need to wrap your code into function expression and return the last num you get. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def sum_digits(num: int) -> int:

    while num > 9:
        num = sum(map(int, str(num)))
        
    return num

def sum_digits(num: int) -> int:

    return sum_digits(sum(map(int, str(num)))) if num > 9 else num
