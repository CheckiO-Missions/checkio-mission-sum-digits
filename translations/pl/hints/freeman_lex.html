Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę zapalić 💡! Chodzi o to, aby przekonwertować podane num do formatu str, a następnie przekonwertować każdą cyfrę z powrotem do formatu int, znaleźć sumę tych cyfr i powtórzyć, jeśli to konieczne.

Brzmi łatwo. Od czego powinienem zacząć?

Maksymalna liczba jednej cyfry to 9. Należy więc powtarzać operacje, aż num stanie się mniejsza lub równa 9. Dla tego rodzaju zadania pętla while jest optymalna. 

while num> 9:
    ...
Ok. Teraz proszę mi pomóc zacząć konwertować podaną liczbę.

Jak wspomniałem wcześniej, najpierw należy przekonwertować bieżącą liczbę na format string. Proszę to zrobić za pomocą wbudowanej funkcji str(). 

str(num)

string jest interpretowana jako sekwencja, więc teraz można iterować po niej i wykonywać operacje na każdym znaku.

Jak przekonwertować każdy znak z powrotem na liczbę całkowitą?

Najprostszym sposobem jest użycie wbudowanej funkcji map() z argumentami int i ciągiem z poprzedniej podpowiedzi. 

map(int, str(num))

Konwertuje ona każdy znak łańcucha na format int i zwraca całość jako obiekt map. Proszę zauważyć, że przy int nie ma nawiasów. Wynika to z faktu, że odwołują się Państwo do funkcji, a nie ją wywołują.

Czuję, że to prawie zrobione, ale potrzebuję więcej wskazówek.

Teraz mają Państwo ciąg cyfr jako obiekt map. Fajną rzeczą jest to, że mogą Państwo użyć tego obiektu jako argumentu wbudowanej funkcji sum() bez dalszej konwersji! 

sum(map(int, str(num)))

W rzeczywistości jest to Państwa nowy obiekt num, więc proszę przypisać wynik obliczeń do zmiennej wewnątrz pętli while.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć swój kod w wyrażenie funkcyjne i zwrócić ostatnią otrzymaną wartość num. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

def sum_digits(num: int) -> int:

    while num> 9:
        num = sum(map(int, str(num)))
        
    return num

Jasne, proszę łapać!

Logika jest taka sama, ale zamiast while używana jest rekurencja pętli. Proszę wywołać tę samą funkcję z przekonwertowanym argumentem. 

def sum_digits(num: int) -> int:

    return sum_digits(sum(map(int, str(num)))) if num> 9 else num

Wiele zadań można łatwo rozwiązać, jeśli zna się matematykę! Suma cyfr jest równa wynikowi dzielenia modulo, jeśli nie jest równa zero, w przeciwnym razie 9. 

def sum_digits(num: int) -> int:

    return num and (num % 9 or 9)
